# NyaySathi Legal Guidance Website

## Overview
NyaySathi is a dedicated platform designed to provide legal guidance and resources to individuals seeking information about their legal rights and responsibilities.

## Features
- Comprehensive legal advice and guidance.
- User-friendly interface for easy navigation.
- Database of legal documents.
- Community forum for discussions and advice.

## Tech Stack
- **Frontend:** React, HTML, CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB

## Setup Instructions
1. **Clone the repository:**  
   `git clone https://github.com/arjunkadamcomp2024-sudo/NyaySathi.git`
2. **Navigate to the directory:**  
   `cd NyaySathi`
3. **Install dependencies for the backend:**  
   `npm install` in the `backend` directory.
4. **Start the backend server:**  
   `node index.js`
5. **Install dependencies for the frontend:**  
   `npm install` in the `frontend` directory.
6. **Start the frontend:**  
   `npm start`
